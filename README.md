# GitBeokTree

1. 유저(github)의 Repository 보기
2. 특정 Repository 폴더 구조 보기
3. 특정 파일 보기

  - LifeCycle
  - LiveData
  - Navigation
  - Paging
  - Room
  - ViewModel
- Network
  - Retrofit
- Async
  - Coroutine
  - RxJava
- DI
  - Koin
